# Presentation Clock (PowerPoint Add-On)

![PresentationTimer](https://github.com/user-attachments/assets/672cb840-b189-4f28-8b23-1e86aad48385)

## Build

* Open project in visual studio
* Select debug or release
* Select build solution
* Optionally select deploy to make installer

## Install

* Use installer setup executable
* OR Double click the VSTO file to install the addon
    * this second option requires that you do not delete or move the add on files though

## Run

* Open your Powerpoint project
* Navigate to the Slide Show tab
* Go to the Timer menu group, and click Presentation Clock button
